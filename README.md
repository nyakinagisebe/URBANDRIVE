# Project Setup

This repository contains the codebase for the project. Due to GitHub's file size limitations, certain large datasets and configuration files could not be pushed directly to the remote repository. 

To run the project successfully, you must download these external assets and place them into the correct directory(Module1 sub-folder).

## Required Large Files

The link provided in the submission text contains the following essential files:
1. **`investment_infrastructure.json`** — The infrastructure configuration and investment data.
2. **`map.osm`** — The OpenStreetMap geospatial network data used for spatial analysis and routing.

Link to download the files - https://drive.google.com/drive/u/1/folders/1fGTmoh9kUFqVk7x4M7jwQQVxEVSGgh77
